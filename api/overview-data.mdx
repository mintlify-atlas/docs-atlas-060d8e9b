## Get Overview

<RequestExample>
```bash cURL
curl http://localhost:2442/api/overview
```
</RequestExample>

Retrieves comprehensive dashboard overview data including system metrics, array status, notifications, and UPS information.

**Authentication:** Requires active server configuration
